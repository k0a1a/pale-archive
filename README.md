# The Pale Archive

## To install this experimental addon:

To install (Chrome/Chromium):
 - Download https://github.com/k0a1a/pale-archive/archive/refs/heads/main.zip
 - Unzip main.zip
 - Go to chrome://extensions → enable Developer Mode → Load unpacked → select `pale-archive-addon` folder

## To run the Archive

To run (Chrome/Chromium):
 - Click Extensions icom in the toolbar (puzzle piece)
 - Click on Pale Archive
 - Click "Run"
